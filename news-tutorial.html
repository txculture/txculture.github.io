概述

网站所有内容修改指南

📌 本网站的所有内容都保存在 index.html 这一个文件中，没有额外的数据文件。

📌 用任意文本编辑器（推荐 VS Code）打开 index.html 即可修改。

📌 修改完成后保存文件，刷新网页（Ctrl + F5）即可看到效果。

📌 建议每次修改前先 备份 index.html 文件，改坏了可以恢复。

⚠️ 重要提醒

修改时请注意：英文引号、英文逗号、英文括号 等符号必须使用半角（英文）字符，不能用中文标点！

📁 文件结构

了解网站文件结构，帮助你快速定位需要修改的位置：

📁 网站根目录/

├── index.html ← ⭐ 主页面（所有内容都在这里）

├── style.css ← 共享样式（一般不需要改）

├── about.html ← 跳转页（自动跳转到品牌故事）

└── news-tutorial.html ← 本教程页面

🎯 一、Banner 横幅修改

Banner 是网页顶部的大横幅区域，显示主标题和副标题。搜索 一缕天香 可以定位到。

修改主标题

搜索 一缕天香，找到如下代码：

<div class="banner-title-row">
    <span class="title-text text-white">一缕天香</span>           <!-- ← 修改左侧文字 -->
    <span class="title-divider"></span>
    <span class="title-text gold-accent">匠心传承</span>       <!-- ← 修改右侧文字 -->
</div>

修改副标题

搜索 TianXiang，找到如下代码：

<p class="...banner-subtitle ...">
    Hainan Tianxiang Culture Co., Ltd.    <!-- ← 修改英文副标题 -->
</p>

💡 提示

Banner 的背景是 CSS 绘制的动画效果，不是图片，所以不需要替换背景图。如果需要完全自定义背景，需要修改 CSS 中的 .banner-bg-gradient 样式。

🛍️ 二、核心产品系列（重点）

产品区域由两部分组成：HTML 卡片（首页展示）+ JS 数据（弹窗详情），修改时两部分都要同步更新。

🏗️ 产品系统架构

1️⃣ HTML 卡片区（搜索"核心产品系列"）— 首页上看到的4张大卡片

    每张卡片 = 一个产品系列的入口，点击后弹出详情

2️⃣ JS products 对象（搜索"const products"）— 产品详情弹窗数据

    每个系列下的具体产品列表，包含名称、图片、描述、价格

2.1 产品卡片 HTML 结构

搜索 核心产品系列，往下找到 grid 布局区域。每张卡片结构如下：

<!-- 沉香系列 -->
<div class="fade-in">

         onclick="openModal('aloeswood')">   <!-- ← 卡片ID，必须与JS数据一致 -->
        <img src="http://static.photos/minimal/640x360/88"         <!-- ← 卡片背景图 -->
             alt="沉香系列">                              <!-- ← 图片alt文字 -->

            <p class="text-white/50 ...">沉香手串、熏香、精油、沉香酒</p> <!-- ← 系列简介 -->
        </div>
    </div>
</div>

搜索 const products，找到产品详情数据。每个产品系列结构如下：

        description: '沉香，被誉为...',        // ★ 系列描述（弹窗顶部显示）

            {
                name: '沉香手串',               // ★ 产品名称
                image: 'http://static.photos/...', // ★ 产品图片URL
                desc: '精选百年沉香老料...',  // ★ 产品描述
                price: '¥1,280起'               // ★ 产品价格

            // ... 更多产品

    rosewood: { ... },    // 花梨木系列
    jiangzhen: { ... },    // 降真香系列
    healthwine: { ... }     // 保健酒系列

字段 说明 示例 必填

description系列描述（仅系列级）'沉香，被誉为...'★
items产品列表（仅系列级）数组 [{...}, {...}]★
image产品图片URL完整URL地址★
desc产品描述'精选百年沉香...'★
price产品价格'¥1,280起'★

第二步：添加 HTML 卡片

<!-- 檀香系列 -->

⚠️ 关键：ID 必须一致

HTML 中 onclick="openModal('sandalwood')" 的参数 sandalwood，必须与 JS 中 sandalwood: { ... } 的键名完全一致！否则点击卡片无法弹出详情。

2.4 删除一个产品系列

同样需要删除两处：HTML 卡片 + JS 数据。

1

删除 HTML 卡片

找到对应的 <!-- xxx系列 --> 注释，删除整个 <div class="fade-in">...</div> 块

2

删除 JS 数据

删除对应的整个键值对，如 jiangzhen: { ... }, 注意逗号处理

const products = {
    aloeswood: { ... },
    rosewood: { ... },
//  👇 删除这一整段 ──────────────────
//  jiangzhen: { ... },
//  👆 删除到这里 ──────────────────────
    healthwine: { ... }       // ← 注意：这里原来有逗号，删除上一项后逗号要去掉或保留
};

🔴 删除注意事项

删除 JS 对象中的一个键值对时，注意前后的逗号。如果删除的是最后一项，前一项末尾的逗号要去掉（JS 不允许对象末尾有多余逗号虽然现代浏览器可能容忍）。如果删除中间项，确保剩余项之间的逗号正确。

搜索 brandStoryData，找到品牌故事数据对象：

const brandStoryData = {

📰 四、新闻动态修改

搜索 newsData，找到新闻数据数组。每条新闻的数据结构：

字段名 说明 格式/示例 必填

❓ 八、常见问题

Q：修改后网页没有变化？

A：按 Ctrl + F5 强制刷新（清缓存），或用浏览器无痕模式打开。

Q：点击产品卡片弹窗没有内容？

A：检查 HTML 中 onclick="openModal('xxx')" 的参数 xxx 是否与 JS products 对象中的键名完全一致（区分大小写）。

Q：添加新产品系列后卡片排列不对？

A：当前使用4列网格布局（lg:grid-cols-4）。如果产品系列不是4个，可以修改这个数字：
2个系列 → lg:grid-cols-2
3个系列 → lg:grid-cols-3
5个系列 → lg:grid-cols-5 或保持 lg:grid-cols-4 自动换行

Q：图片显示不出来？

A：① 检查图片URL是否正确，用浏览器直接打开URL测试；
② 确保URL用英文引号包裹，没有多余空格；
③ 如果是本地图片，检查文件路径和大小写。

Q：不小心改坏了怎么办？

A：修改前建议先备份 index.html 文件。改坏了用备份恢复即可。也可以用 VS Code 的 Ctrl + Z 撤销功能。

Q：日期格式写错了会怎样？

A：新闻列表中日期显示会异常。必须使用 YYYY-MM-DD 格式，如 2025-07-15。

Q：新闻列表显示几条？

A：列表默认显示最新5条。点击"查看更多"可看全部。数据量不限。

Q：代码中引号用了中文的会怎样？

A：会导致 JavaScript 报错，整个页面功能可能失效！请务必使用英文半角引号 '...' 或 "..."，不要使用 '…' 或 "…"。

Q：如何修改导航栏菜单项？

A：搜索 首页，找到导航区域。桌面导航和移动端菜单是两套，需要同时修改。导航链接指向页面内锚点（如 #products），对应各区域的 id 属性。

Q：如何修改网站图标（Favicon）？

A：在 <head> 中找到 <link rel="icon" ...>，替换 href 为你的图标文件路径即可。
